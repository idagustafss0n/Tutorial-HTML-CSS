Information om HTML

HTML grunder

Ett html dokument består av <head> och <body>. Mellan Head skrivs information som inte syns på sidan, till exempel sökoptimering ligger där. Även bild och text som syns på webbläsar-fliken står i head.

Mellan Body ligger all kod som sedan ska synas på hemsidan.Bland annat text, bilder, formulär, tabeller.

Både head och body ligger mellan <html>

Klasser och ID

Klasser och ID går att använda i htmk filen för att grupera eller separera ett textstycke, box eller liknande för att kunna styla detta i css.

Klasser betecknas med class=XX i html-filen och .XX i css-filen. En klass går att använda flera gånger under samma dokument.

ID betecknas med id=XX i htmk-filen och #XX i css-filen. Id går bara att användas på ett enskilt element.

Vid en konflik mellan klass, id och typselektor (bland annat p och h-taggar) så vinner id, sen klass och sist typselektor
