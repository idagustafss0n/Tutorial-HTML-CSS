Text, font, färg och storlek

Genom css går det att ändra font, storlek och färg på text.

Genom elementet "span" går det att ändra text direkt i ett annat element. Vi demonstrerar det via text-decoration. Text-decoration gör att vi kan ge texten överstruktet , understruket , genomstruket och alla tre

Padding & margin

Texten är innehållet i boxen, mellan innehåll och kanten finns padding. Sen kommer kanten (svart färg). utanför kanten finns marginalen.

Positioner

Det finns fem olika positioner inom css.

  • static
  • relative
  • fixed
  • absolute
  • sticky

Med position static lägger sig innehållt exakt där det skulle lägga sig även utan formatering.

Med position relative flyttar du det påverkade innehållet till önskat ställe men med det ursprungliga stället som utgångsläge.

Med position fixed väljer du hur ett visst element ska placeras i förhållande till webbläsarfönstret, fixerat. Rutan nere i högra hörnet är placerade med fixed

Med position absolute kan du placera ett element på valfritt ställe på webbsidan. Den absoluta positionering är relativt föräldraelementet.

Med position sticky fastnar elementet i toppen av sidan när man scrollar. Rutan nedanför är är sticky 20px ovanför botten.
